### Age In Minutes

---

안드로이드 코틀린 적용을 위한 소소한 연습 프로젝트. 

태어난 일자를 입력하면 현재까지 일자를 계산하여 보여준다.

## # 개요

---

- Chapter1. 프로젝트 생성.
- Chapter2. UI 세팅 및 완성.
- Chapter3. DatePickerDialog 추가.
- Chapter4. 선택 날짜 반영 및 SimpleDateFormat 날짜 확인
- Chapter5. 분 단위 나이 계산기 완성
- Chapter6. 일자별 나이 계산기 완성

## # 소개

---

- 안드로이드 개인 토이프로젝트
- 개발언어 : Android(Kotlin)
- 소스코드 : Github
